### List Credit Lines

<ApiEndpoint method="POST" path="/api/v4/credit-line/loans/info" />

This endpoint returns an active loan.

***Please, pay attention that this endpoint works on demand. It means that you need to contact WhiteBIT support and provide your API key to get access to this functionality.***

***You can also request access via the form [here](https://institutional.whitebit.com/portfolio-margin-access).***

‚ùó Rate limit 1000 requests/10 sec.

**Response is cached for:**
NONE

**Response:**

Available statuses:

- `Status 200`

```json
{
  "asset": "BTC",
  "amount": "1.5",
  "totalFunding": "0.5",
  "currentLtv": "25",
  "initialLtv": "60",
  "marginCallLtv": "75",
  "liquidationLtv": "85",
  "createdAt": 1715339355
}
```
- `Status 400 if no information available about active loan`

```json
{
  "code": 0,
  "message": "No active loan at the moment"
}
```

- `Status 404 if loans not available. Please, contact support to enable this feature for your account.`

```json
{
  "code": 0,
  "message": ""
}
```

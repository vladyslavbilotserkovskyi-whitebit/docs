### Get Flex Payment History

<ApiEndpoint method="POST" path="/api/v4/main-account/smart-flex/investments/payment-history" />

Retrieve investment earnings history (ONLY DAILY_EARNING operations).

- Note: This endpoint automatically filters to show ONLY DAILY_EARNING operations (type 4)

‚ùó Rate limit 1000 requests/10 sec.

**Response is cached for:**
NONE

**Parameters:**

| Name        | Type    | Mandatory | Description                   |
|-------------|---------|-----------|-------------------------------|
| limit       | Integer | **No**    | Pagination limit. Default: 50.|
| offset      | Integer | **No**    | Pagination offset. Default: 0.|
| plan        | String  | **No**    | Filter by plan ID (UUID).     |
| investment  | String  | **No**    | Filter by investment ID.      |
| transaction | String  | **No**    | Filter by transaction ID.     |
| dateFrom    | Integer | **No**    | Filter from date (timestamp). |
| dateTo      | Integer | **No**    | Filter to date (timestamp).   |

**Request BODY raw:**

```json
{
  "limit": 50,
  "offset": 0,
  "plan": "8f2e9d3c-1a4b-4c2d-9e5f-6a7b8c9d0e1f",
  "investment": "inv_123",
  "transaction": "tx_456",
  "dateFrom": 1640995200,
  "dateTo": 1641081600,
  "request": "{{request}}",
  "nonce": "{{nonce}}"
}
```

**Response:**

Available statuses:

- `Status 200`

*Same format as investment history, but automatically filtered to show ONLY DAILY_EARNING operations (actionType = 4).*

```json
}
```

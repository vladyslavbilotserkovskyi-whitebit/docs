### Get Mining Account Hashrate

<ApiEndpoint method="POST" path="/api/v4/mining/hashrate" />

This endpoint returns hashrate of mining pool account

‚ùó Rate limit 1000 requests/10 sec.

**Response is cached for:**
NONE

**Parameters:**

| Name      | Type    | Mandatory | Description                                                                                                                                                                                         |
|-----------|---------|-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| account   | String  | **Yes**   | Mining pool account                                                                                                                                                                                 |
| from      | Int     | **No**    | Unix timestamp of starting point                                                                                                                                                                    |
| to        | Int     | **No**    | Unix timestamp of final point                                                                                                                                                                        |
| interval  | String  | **No**    | Timestamp interval (5m, 1h, 24h). Default: 1h                                                                                                                                                        |

**Request BODY raw:**

```json
{
  "account": "my-mining-account-01",
  "from": 1717671600,
  "to": 1717675200,
  "interval": "1h"
}
```

**Response:**

Available statuses:

- `Status 200`
- `Status 422 if request validation failed`

```json
{
  "data": {
      "account": "my-mining-account-01",
      "hashrate": [
         {
            "timestamp": 1717671600,
            "hashrate": "123.4533333333333333",
         },
         {
            "timestamp": 1717675200,
            "hashrate": "322.4533333333333333",
         }
      ]
  }
}
```

<details>
<summary><b>Errors:</b></summary>

```json
{
  "code": 422,
  "message": "Incorrect timestamp"
}
```

</details>

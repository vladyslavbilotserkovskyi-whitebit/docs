### Get User Flex Investments

<ApiEndpoint method="POST" path="/api/v4/main-account/smart-flex/investments" />

Retrieve user's investment portfolio with optional filtering.

‚ùó Rate limit 1000 requests/10 sec.

**Response is cached for:**
NONE

**Parameters:**

| Name             | Type    | Mandatory | Description                                       |
|------------------|---------|-----------|---------------------------------------------------|
| limit            | Integer | **No**    | Pagination limit. Default: 100.                   |
| offset           | Integer | **No**    | Pagination offset. Default: 0.                    |
| ticker           | String  | **No**    | Filter by currency ticker. Example: USDT.         |
| plan             | String  | **No**    | Filter by plan ID (UUID).                         |
| investment       | String  | **No**    | Filter by investment ID.                          |
| investmentStatus | Integer | **No**    | Filter by status (1=ACTIVE, 0=CLOSED).            |

**Request BODY raw:**

```json
{
  "limit": 100,
  "offset": 0,
  "ticker": "USDT",
  "plan": "8f2e9d3c-1a4b-4c2d-9e5f-6a7b8c9d0e1f",
  "investment": "invest_id_123",
  "investmentStatus": 1,
  "request": "{{request}}",
  "nonce": "{{nonce}}"
}
```

**Response:**

Available statuses:

- `Status 200`

```json
{

      "status": 1,

    }
  ],
  "limit": 100,
  "offset": 0
}
```

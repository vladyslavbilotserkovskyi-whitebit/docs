### Get Flex Plans

<ApiEndpoint method="POST" path="/api/v4/main-account/smart-flex/plans" />

Retrieve list of active [Flex Plans](./../glossary.md#crypto-lending).

‚ùó Rate limit 1000 requests/10 sec.

**Response is cached for:**
NONE

**Parameters:**

| Name   | Type    | Mandatory | Description                                  |
|--------|---------|-----------|----------------------------------------------|
| limit  | Integer | **No**    | Pagination limit. Default: 100, Max: 100.    |
| offset | Integer | **No**    | Pagination offset. Default: 0, Min: 0.       |
| ticker | String  | **No**    | Filter by currency ticker. Example: USDT.    |

**Request BODY raw:**

```json
{
  "limit": 50,
  "offset": 0,
  "ticker": "USDT",
  "request": "{{request}}",
  "nonce": "{{nonce}}"
}
```

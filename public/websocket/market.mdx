### Market statistics

```json

  "id": 5,
  "method": "market_request",
  "params": [
    "ETH_BTC", // market
    86400 // period in seconds
  ]
}
```

##### ‚§µÔ∏è Response

```json

  "id": 5,
  "result": {
    "period": 86400, // period in seconds
    "last": "0.020981", // last price

  "error": null

```

#### Subscribe

You can subscribe only for 86400s (24h from now).

Update interval: 1 sec

##### ‚§¥Ô∏è Request

```json
{
    "id": 6,
    "method": "market_subscribe",
    "params": [
        "ETH_BTC", // markets
        "BTC_USDT",
    ]
}
```

##### ‚§µÔ∏è Response

```json
{
  "id": 6,
  "result": {
    "status": "success"
  },
  "error": null
}
```
